# Dippy

A solidity contract compiler modeled after [Remix](https://remix.ethereum.org/).
View it live [here](http://dj-mc.herokuapp.com/).

### RUN
Clone or fork the repository.

```bash
$ npm i -g ethereumjs-testrpc
$ npm i
```
```bash
$ testrpc

# in another terminal tab
$ npm start
```

# Dj Dippy

A solidity contract compiler modeled after [Remix](https://remix.ethereum.org/).

![Go DJ!!](assets/godj.png)

### RUN
Clone or fork the repository.

```bash
$ npm install --global ganache-cli
$ npm install
```
```bash
$ ganache-cli -u 0

# in a new terminal window
$ npm start
```
